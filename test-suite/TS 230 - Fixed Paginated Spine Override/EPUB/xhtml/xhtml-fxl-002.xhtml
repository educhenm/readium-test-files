[REQUIRED] fxl-230 Fixed layout support

Tests whether setting the itemref property rendition:flow-* property to flow-scrolled-doc or flow-scrolled-continuous has any effect. The file declares its layout to be pre-paginated and the rendition-flow to be paginated. Per spec, pre-paginated layout cannot be overridden with rendition:flow-*, so the overrides should be ignored. So all chapters should be paginated.

If all spine items are paginated, then the test passes.
