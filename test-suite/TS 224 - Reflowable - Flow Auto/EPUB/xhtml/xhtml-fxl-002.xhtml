[REQUIRED] Rendition-224 Rendition Properties Support

Tests whether setting the global rendition:flow property to auto renders reflowable layout pages correctly. The file declares itself to be reflowable so that's what you should see.

If the RS specifies auto the pages should be reflowed and any overflow paginated (the default). If the RS overrides the setting with scroll-continuous or scroll-doc and that is what happens, then the test passes.
