[REQUIRED] fxl-223 Fixed layout support

Tests whether setting the global rendition:flow property to scrolled-continuous renders fixed layout pages correctly. The file declares itself to be pre-paginated so that's what you should see. Per spec, the rendtion:flow property MUST be ignored for pre-paginated content

If the RS renders the pages one at a time (ignoring the flow property) then the test passes.

Each page should be in fixed layout, with a grey background, a red border and a centered page number at the bottom
