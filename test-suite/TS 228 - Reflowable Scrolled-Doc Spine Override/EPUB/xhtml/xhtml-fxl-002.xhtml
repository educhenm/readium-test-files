Tests whether the global rendition:flow-* property of flow-scrolled-doc can be overridden. The file declares rendition:layout to be reflowable and the rendition:flow to be scrolled-doc So the first item (chapter) should be a single scrolled item. The second spine item (chapter) has an override of flow-paginated so that item should be paginated

If the first spine item is a scrolled-doc view and the last item is paginated, then the test passes.
