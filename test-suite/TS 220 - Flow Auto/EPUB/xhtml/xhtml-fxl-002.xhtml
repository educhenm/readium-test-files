Tests whether setting the global rendition:flow property to auto so the pages should be laid out as one page at a time. The file declares itself to be pre-paginated so that's what you should see.

If the RS user settings specify auto then the pages should be paginated. Per spec, the rendtion:flow property MUST be ignored for pre-paginated content

Each page should be in fixed layout, with a grey background, a red border and a centered page number at the bottom
