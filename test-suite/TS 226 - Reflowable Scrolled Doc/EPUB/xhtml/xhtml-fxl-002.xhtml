[REQUIRED] Rendition-226 Rendition Properties Support

Tests whether setting the global rendition:flow property to scrolled-doc renders reflowable layout pages in a single scrollable window. The file declares itself to be reflowable so that's what you should see.

If the RS user-settings specify auto the pages should be reflowed and any overflow rendered as a scrollable doc for that spine item only. If the RS overrides the setting with paginated or scroll-doc-continuous and that is what happens, then the test passes.
