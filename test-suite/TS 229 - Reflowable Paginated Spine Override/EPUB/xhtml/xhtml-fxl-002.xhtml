[REQUIRED] Rendition-229 Rendition Properties Support

Tests whether the global rendition:flow property of paginated can be overridden. The file declares rendition:layout to be reflowable and the rendition:flow to be paginated. So the first item (chapter) should be a series of pages. The second spine item (chapter) has an override of flow-scrolled-doc so that item should be a single scrollable view

If the first spine item is paginated and the last item is a scrollable view, then the test passes.
